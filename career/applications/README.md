# Applications Workspace

Use this folder for company-specific material.

## Suggested structure

```text
career/applications/
  company-name/
    role-brief.md
    tailored-resume-notes.md
    cover-letter.md
    outreach.md
    interview-notes.md
```

## Practical workflow

1. Duplicate the template for the target company.
2. Pull the most relevant projects from the dashboard.
3. Tailor the resume summary and featured case studies.
4. Draft the cover letter from the closest matching template.
5. Add follow-up and interview notes in the same folder.
