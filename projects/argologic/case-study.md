# Overview

Argologic was a web project from the DRKZEN agency period. The brief called for a clean, well-structured site that communicated the product or service clearly and without friction — a typical agency deliverable that required both design thinking and implementation capability.

# What I did

- Directed the overall site UI and visual approach
- Provided frontend implementation support across the build
- Collaborated on layout, information structure and content presentation

# Outcome

This project represents the kind of agency engagement where design direction and frontend implementation were delivered in the same role — an uncommon combination that reflects the broader skill set developed through the DRKZEN studio period. The site demonstrated that clean, purposeful UI work does not require a separate handoff between design and code.
